# Codex instructions

Before doing any work, read [`CLAUDE.md`](./CLAUDE.md) and follow it as the
repository-wide source of truth.

Before reading or modifying files, load the applicable additional instructions:

- Files under `tests/`: read [`tests/CLAUDE.md`](./tests/CLAUDE.md).
- `package.json`: read [`.claude/rules/techstack.md`](./.claude/rules/techstack.md)
  and [`.claude/rules/npm-commands.md`](./.claude/rules/npm-commands.md).
- Files under `src/routes/` or `src/mcp/`: read
  [`.claude/rules/logging.md`](./.claude/rules/logging.md).
- Files under `src/mcp/`: read
  [`.claude/rules/mcp-tools.md`](./.claude/rules/mcp-tools.md).
- Files under `src/style/`: read
  [`.claude/rules/styles.md`](./.claude/rules/styles.md).
- A directory `CLAUDE.md` or a file under `.claude/rules/`: read
  [`.claude/rules/agent-rules.md`](./.claude/rules/agent-rules.md).

Instructions in the more specific file take precedence when they conflict.
